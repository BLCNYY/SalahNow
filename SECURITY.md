# Security Policy

## Supported versions

Security fixes will be applied to the `main` branch. Please ensure your fork is up to date before reporting issues.

## Reporting a vulnerability

If you discover a security vulnerability, email contact@blcnyy.dev with:
- Steps to reproduce the issue
- Expected versus actual behavior
- Any relevant logs or screenshots

You will receive an acknowledgement within five business days. Please do not open public issues for security reports until a fix is released.
